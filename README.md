# 📚 AI Study Assistant (LLM Powered)

An **AI-powered Study Assistant** that answers academic questions using a **Large Language Model (LLM)**.
Users can ask questions and choose the **response style (Friendly or Academic)**.
The application generates answers using the **Google Generative AI API**.

This project demonstrates **Python + LLM integration + prompt engineering** to build an intelligent study tool.

---

# 🚀 Features

* 🤖 AI powered answers using **Google LLM**
* 🎓 Academic or Friendly response styles
* ⚡ Instant response generation
* 🧠 Prompt engineering based responses
* 🌐 Simple and interactive interface
* 🐍 Built completely with **Python**

---

# 🛠️ Tech Stack

* Python
* Google Generative AI
* LLM (Large Language Model)
* Gradio (UI)
* Prompt Engineering

# ⚙️ Installation

## 1 Clone the repository

```
git clone https://github.com/your-username/study-assistant.git
cd study-assistant
```

---

## 2 Create virtual environment

```
python -m venv venv
```

Activate environment

Mac / Linux

```
source venv/bin/activate
```

Windows

```
venv\Scripts\activate
```

---

## 3 Install dependencies

```
pip install -r requirements.txt
```

---

# 🔑 Setup Google API Key

Create a `.env` file in your project root and add:

```
GOOGLE_API_KEY=your_google_api_key_here
```

Get your API key here:

https://makersuite.google.com/app/apikey

---

# ▶️ Run the Application

Run the following command:

```
python app.py
```

The application will start and open in your browser.
